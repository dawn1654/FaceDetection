# Face Detection using OpenCV

This project performs real-time face and eye detection using Python and OpenCV.
The program captures video from the webcam and detects faces and eyes using Haar Cascade classifiers.

## Features

* Real-time webcam face detection
* Eye detection inside detected faces
* Uses OpenCV Haar Cascade classifiers
* Simple and lightweight implementation

## Technologies Used

* Python
* OpenCV

## Requirements

Install the required library:

pip install opencv-python

## Running the Program

Clone the repository:

git clone https://github.com/YOUR_USERNAME/FaceDetection.git

Navigate to the project folder:

cd FaceDetection

Run the script:

python main.py

## How It Works

The program uses Haar Cascade classifiers provided by OpenCV to detect facial features.
Each frame from the webcam is converted to grayscale and processed to identify faces and eyes.
